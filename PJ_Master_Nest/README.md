## About this course

Learn to develop and test enterprise grade Node.js applications in TypeScript.  
Learn modern workflows using Docker.  


## Environment Setup


- install NestJS  
	```sh
	$ npm i -g @nestjs/cli
	```

- create NestJS project  
	```sh
	$ nest new project-name
	```

- install Postman  
	```sh
	$ sudo snap install postman
	```
